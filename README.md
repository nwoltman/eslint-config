# @nwoltman/eslint-config

My personal, highly specific ESLint config.


## Installation

```sh
# npm
npm install @nwoltman/eslint-config

# yarn
yarn add @nwoltman/eslint-config
```


## Usage

**.eslintrc.json**

```js
{
  "extends": "@nwoltman/eslint-config",
  "rules": {
    // Your rules here
  }
}
```
